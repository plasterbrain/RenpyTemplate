  • Select choice - Enter, left click
  • Rollback - PgUp, scroll up

Accessibility Features

{Your Game} includes some features designed to make the game more accessible. You can toggle them in the Settings menu.

  • Change the game's default font to DejaVu Sans or OpenDyslexic
  • Adjust text box opacity
  • Adjust text size and line spacing
  • Enable subtitles for important sounds
  • Toggle self-voicing or clipboard voicing mode

Self-Voicing Mode

When self-voicing mode is enabled, in-game text will be read aloud using the default speech synthesis software for your operating system. Self-voicing mode can be toggled either by pressing v, Shift + V, or through the preferences menu.

You can use the keyboard arrows to navigate through and interact with any focusable elements on screen. When an element is focused, the game will announce the text associated with that element. When no element is focused, the game will announce whatever non-focusable text is on screen, such as the dialogue.

Self-voicing mode is not supported on Android, iOS, or Chrome OS.

Speech Settings

Windows

Self-voicing on Windows is handled by the Microsoft Speech API. Reading voice and speed can be changed by going to Control Panel, Ease of Access, Speech Recognition, and then selecting Text to Speech. This will open a window called Speech Properties where you can adjust voice settings.

Mac

Self-voicing on Mac is handled through the "say" command. Speech settings can be adjusted by going to System Preferences. They are located in different locations depending on your OS version:

  • 10.6 (Snow Leopard) and 10.7 (Lion) Under Speech, then the Text to Speech tab.
  • 10.8 (Mountain Lion) through 10.11 (El Capitan): Dictation & Speech, then the Text to Speech tab.
  • 10.12 (Sierra) and newer: Accessibility, then the Speech tab.
Linux

Self-voicing on Linux is handled using eSpeak. If self-voicing is not working on your system, you may need to install eSpeak using your package manager.

Clipboard Voicing Mode

If you are using a screenreader that can process text copied to the clipboard, you may prefer to use clipboard voicing mode. Clipboard voicing can be toggled by pressing Shift + C or through the accessibility menu.

In clipboard voicing mode, when an element is focused using the keyboard or mouse, any text currently associated with that element will be copied to the clipboard. If no element is focused, any non-focusable text on screen, such as dialogue, may be copied to the clipboard instead.

Uninstalling the Game

Steam

  1. Find and click on {Your Game} in the Steam client.
  2. Click the gear icon on the right side. Then under "Manage," click Uninstall.
  3. Click Uninstall in the popup window that appears to confirm your selection.

Itch.io

If you downloaded the game from itch.io directly, you can simply delete the game folder to remove it from your computer.

If you are using the app:

  1. Open the Itch app and select the Library tab.
  2. Find {Your Game} under "Installed items." You may need to select View all... on the right if you have many games installed.
  3. Right click on the {Your Game} thumbnail and select Uninstall.
  4. Click Uninstall now in the popup that appears to confirm your selection.

Removing Save Data

{Your Game} stores save files in the app data folder. You'll have to delete this folder separately if you want to completely remove the game from your system.

Windows
%USERPROFILE%/AppData/Roaming/RenPy

Mac
~/Library/RenPy

Linux
~/.renpy

For Itch app users, the user folder on Windows containing your save data may be a specially created user beginning with "itch-player-" if you ran the game in Sandbox Mode.

Performance Issues

You may experience lag if your graphics drivers are out of date or if your system does not adequately support hardware-accelerated graphics.

  1. Ensure your graphics drivers are up to date.
  2. Select a new renderer in the Settings menu under Graphics. "Auto" will select a renderer best suited for your system, but you may want to try selecting "OpenGL" and finally "ANGLE/DirectX" if you are experiencing lag. Restart the game after changing renderers.
  3. For low-end systems, videos and animated transitions can be disabled from the Graphics settings screen.
  4. If playing from a laptop, you may experience better performance by keeping your computer connected to power or optimizing battery settings for performance rather than battery life.

Failing to Launch

There are a number of reasons the game may fail to start on your system:

Graphics drivers
In rare cases, an outdated or buggy graphics card driver will cause the game to not launch. Make sure your drivers are up to date before running the game.
File path encoding issues
On Windows systems, the game may not start correctly if it relies on file paths containing characters outside the system's language. You will need to adjust your system locale for non-unicode programs, either through Windows settings or by using a system locale utility such as Locale Emulator.
Running on a non-primary display
On Mac OS X 10.9 (Mavericks), you will need to launch the game from the primary display to ensure it starts correctly.
